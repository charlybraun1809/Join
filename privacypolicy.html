  • Correction of inaccurate personal data (Art. 16 GDPR),

Cookie Usage

A cookie is a small data file created when you visit a website and temporarily stored on the system of the website visitor. If the server of this website is accessed again by the user, the browser of the user sends the previously received cookie back to the server.

Detailed information about cookies used on this website can be accessed in the Cookie Settings.
